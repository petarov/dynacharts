DynaCharts Server
===================

Java HTTP Server Backend (Deprecated)

## Requirements

  * Java JDK 1.7.x
  * Apache Maven 3.2.x
  * Eclipse Kepler (4.3.2) 

## Usage
To clean and compile the project use:

    mvn clean

To deploy the app within Maven-Jetty server plug-in use:

    mvn jetty:start
